# Curso de Fundamentos de .NET - .NET Clean Architecture

Este repositorio contiene mi implementación del proyecto desarrollado durante el curso de fundamentos de .NET impartido por **Héctor de León**. 

El objetivo principal de este proyecto es meramente educativo: comprender las bases del ecosistema .NET y aprender a estructurar aplicaciones siguiendo los principios de la **Arquitectura Limpia (Clean Architecture)**.

## 🛠️ Tecnologías utilizadas
* .NET 10
* C#

## 📝 Créditos y Referencias
* **Instructor:** Héctor de León 
* **Curso:** Curso de Fundamentos de .NET
